# Spacemacs
When using a new editor with lots of new key bindings, keeping this page open can be quite handy.

# Install
`npm i -g elm-app`  
`npm i`  
`elm-package install`  

# Start / Develop mode
`elm-app start`

# Build
Results in a package in the `dist` folder with all the statics bundled.
`elm-app build`
